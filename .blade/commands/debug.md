## 分析步骤

- 语法错误 / 运行时错误 / 类型错误 / 逻辑错误？
- 来自哪个模块？

### 2. 定位源头
- 从错误堆栈追溯
- 找到触发错误的代码

### 3. 上下文分析
- 阅读相关代码
- 理解预期行为
- 找出实际行为与预期的差异

### 4. 根因分析
- 为什么会发生这个错误？
- 什么条件触发了它？

### 5. 修复建议
- 提供具体的修复方案
- 说明修复原理
- 考虑边界情况
